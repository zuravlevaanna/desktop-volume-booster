# Главное приложение (DesktopApp)

Здесь лежит всё, что касается внешнего вида программы, которую пользователь устанавливает на компьютер.

### Что тут будет:
* **Интерфейс:** Окно с ползунком громкости, кнопками и красивыми индикаторами звука.
* **Управление в трее:** Маленькая иконка рядом с часами, чтобы быстро менять громкость.
* **Настройки:** Здесь программа запоминает, какие уровни звука выбрал пользователь.

### Как это работает:
Приложение забирает настройки из облака (через ServerApi) и передает команды звуковому движку, чтобы тот делал звук громче.

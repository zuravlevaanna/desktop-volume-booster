# SoundMaster: Усилитель звука
Простая и мощная программа, которая делает звук на компьютере громче в 5 раз, не превращая его в шум.

## Где что лежит
* `DesktopApp` — сама программа для Windows/Mac.
* `ServerApi` — сервер для хранения настроек и профилей.
* `Web` — сайт, где можно скачать приложение.

## Технический момент
Для расчёта усиления используем формулу:
$$A_{dB} = 20 \cdot \log_{10}(\frac{V_{out}}{V_{in}})$$

# Серверная часть (ServerApi)

«Мозги» в интернете, которые помогают приложению работать на разных устройствах.

### Что тут будет:
* **Личный кабинет:** База данных, где хранятся логины и пароли пользователей.
* **Облако настроек:** Если ты настроил звук на одном ПК, на другом всё подтянется автоматически.
* **Обновления:** Сервер проверяет, вышла ли новая версия программы.

### Как это работает:
Служит мостиком между базой данных и самим приложением. Общается с помощью простых текстовых запросов (JSON).

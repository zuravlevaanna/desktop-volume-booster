# Сайт проекта (Web)

Это лицо проекта в интернете.

### Что тут будет:
* **Страница загрузки:** Красивое описание программы и большая кнопка «Скачать».
* **Демо-версия:** Плеер, где можно нажать кнопку и услышать, как работает усилитель прямо в браузере.
* **Помощь:** Инструкции для пользователей, если у них что-то не работает.

### Как это работает:
Сайт показывает статистику скачиваний, которую берет у ServerApi, и помогает новым людям узнать о приложении.
